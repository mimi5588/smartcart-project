# SmartCart Design System

## Colors

| Role       | Hex     | Usage                       |
| ---------- | ------- | --------------------------- |
| Primary    | #6FD3C1 | Main buttons and highlights |
| Secondary  | #B8A6FF | Secondary cards and icons   |
| Accent     | #8ED2FF | Links and active elements   |
| Background | #F7F9FC | Main app background         |
| Text       | #1F2937 | Main text color             |
| Error      | #FF7A7A | Error messages              |
| Success    | #63D39B | Success messages            |
